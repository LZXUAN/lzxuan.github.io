SINMIN2U

A Blogger blog created with love for SJK(C) SIN MIN, SIMPANG

BACKGROUND

It all started with my mother's request. My mother is one of the teachers at that school. After noticing other schools have their own website, my mother decided to create one. The problems are my mother doesn't quite know how to make a responsive blog and doesn't have the time to set up a blog. Knowing I am interested, my mother then give me a task to create a blog for her school.

GOALS

  • Show the information of the primary school
  • Update the latest news of the primary school

METHODOLOGY

In this case, I choosed to use Blogger because it is easy to manage and it aces server uptime test. In fact, it is the perfect platform for beginner. But I would recommend WordPress for more advanced bloggers because it is more customizable. 

Setting up a Blogger account is very easy. Just a few simple steps required. Next task, choose a template you like. Honestly, I don't like any templates Blogger provided. So, I decided to take a search for a custom template. Finally, I found the right template, Personal by ThemeXpose. 

After applying the new template, I feel something not right. So, I decided to give it a few tweaks.

I changed the max-width of the header to 100% and deleted its padding because I want to display the school's view.

.headerleft {
  display: block;
  margin: 0 auto;
  /* width: 400px; */
  padding: 40px 0 40px 0;
  line-height: 0;
  text-align: center;
  max-width: 100%;
}


Besides, I also changed one of the width of the footer widget area to 100% and deleted its padding and footer3 and footer4 widget area.

#footer div.column {
display: inline-block;
float: left;
width: 100%;
}
@media (min-width:768px) and (max-width:979px){
...
#footer div.column{
width:100%;
margin-right:2px;
}
<div id='footer'>
  <div class='widget-area'>
    <b:section class='column' id='footer2' showaddelement='yes'>
    </b:section>
    <b:section class='column' id='footer3' showaddelement='yes'>
    </b:section>
    <b:section class='column' id='footer4' showaddelement='yes'>
    </b:section>
  </div>
  <!-- /.widget-area-->  
</div>


Then, I found out that there is no changes of colour of the link when my mouse pointer hovers it. So, I added a new line of code to make it changes colour when the mouse pointer hovers it.

a:hover {
  -webkit-transition: all .2s ease;
  -moz-transition: all .2s ease;
  -ms-transition: all .2s ease;
  -o-transition: all .2s ease;
  transition: all .2s ease;
  color: #04B404!important
}


About Chinese font, I added "Microsoft YaHei New", "Microsoft Yahei", "微软雅黑", 宋体, SimSun, STXihei and "华文细黑" to almost every font-family.

font-family: Open Sans,sans-serif, "Microsoft YaHei New", "Microsoft Yahei", "微软雅黑", 宋体, SimSun, STXihei, "华文细黑";

Finally, update the blog and it's all done now. Below is the preview of the blog.
